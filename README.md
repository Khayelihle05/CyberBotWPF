# Cybersecurity Awareness Bot (PROG6221 POE – Part 2)

## Project Overview

This project is a WPF Cybersecurity Awareness Chatbot developed for PROG6221.

The chatbot helps users learn about cybersecurity topics through an interactive graphical interface.

## Features

* WPF GUI chatbot interface
* Audio greeting
* User name personalisation
* Keyword recognition
* Random chatbot responses
* Memory and recall system
* Sentiment detection
* Conversation flow handling
* Cybersecurity awareness education

## Topics Covered

* Password Safety
* Phishing Awareness
* Online Scams
* Privacy Protection
* General Cybersecurity Tips

## Technologies Used

* C#
* .NET 8
* WPF
* Visual Studio 2022
* GitHub

## How To Run

1. Open the solution in Visual Studio 2022.
2. Build the solution.
3. Run the project.
4. Enter your name and begin chatting with the bot.
